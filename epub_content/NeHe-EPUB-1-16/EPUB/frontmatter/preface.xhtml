Preface

This book is an exploration of using WebGL and three.js using the well-known NeHe lessons as basis for the exploration. The book began as a collaboration between Johannes Raida and Ric Wright. Johannes began writing some lessons using the well known OpenGL lessons known as NeHe. He posted these to his website. Ric then joined in and began porting the rest of the lessons to three.js and WebGL.

All 48 lessons have now been ported. This book covers the first 16 lessons. Subsequent volumes cover lessons 17-32 and 33-48.Not all of them have been ported for various reasons. The first lesson, for example, is just an explanation of how to set up OpenGL. We do cover "setting up" WebGL and three.js, but since it is pretty trivial (all the major browsers now support WebGL by default) the first chapter is pretty short. Some of the other lessons cover subjects (e.g. using bitmap fonts) which are unneeded or "just work" in three.js so don't warrant spending time covering.

Ric is the Director of Engineering for the Readium project, which is an open-source foundation dedicated to providing web apps and a SDK for rendering EPUB, the standard format for eBooks. As a demonstration of the capabilities of EPUB and Readium, Ric took the lessons and created these EPUBs. The details of how this book was created are covered in the colophon at the end of the book.

If you have any feedback, suggestions or bug reports about this book, you are strongly encouraged to contact Ric directly.
